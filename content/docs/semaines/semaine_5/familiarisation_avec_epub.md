# Familiarisation avec EPUB

La semaine prochaine, nous étudierons le format EPUB. Afin de préparer ces travaux, nous vous invitons à faire les lectures préparatoires suivantes :

1. [La page Wikipedia sur les livres électroniques](https://fr.wikipedia.org/wiki/Livre_num%C3%A9rique)
2. [La page Wikipedia sur EPUB](https://fr.wikipedia.org/wiki/EPUB_(format))
3. [La page Wikipedia sur Projet Gutenberg](https://fr.wikipedia.org/wiki/Projet_Gutenberg)

Par ailleurs, vous devriez visiter le site du projet Gutenberg et consulter quelques livres électroniques.
